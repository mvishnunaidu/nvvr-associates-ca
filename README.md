# 🏛️ N V R R & ASSOCIATES — Chartered Accountants Website

A premium, highly responsive, and feature-rich single-page web application designed for **M/s. N V R R & ASSOCIATES**, a registered Chartered Accountancy firm. This project has been optimized for performance, search engine indexing (SEO), and local search visibility to serve clients across India.

## 📈 Search Engine Optimization (SEO) Highlights

To ensure maximum organic search engine traffic, the website includes:
1. **Canonical URLs**: Pointing to `https://nvvrassociates.in` to prevent duplicate indexing issues.
2. **Schema.org Structured Data**: An `AccountingService` schema defining location, phone, address, pricing range, and operational hours.
3. **Optimized Keywords**: Meta tag keywords targeted toward local searches like *"CA in Tirupati"*, *"GST registration AP"*, and *"Tax Consultant Tirupati"*.

## 💻 Local Installation & Development

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/mvishnunaidu/nvvr-associates-ca.git
   ```
2. Navigate to the project directory:
   ```bash
   cd "NVRR & Associates"
   ```
3. Open `index.html` in any web browser (Chrome, Edge, Firefox, Safari) or use VS Code's **Live Server** extension.
